Product Discount

This module allows you to set discounts on products as a percentage. You can set a discount from particular products.

To set discount Go to Products -> General Information -> Set Discount (%).

Show in Website

Discount added products are displayed with discounted value and other products shows 0 as value for discount. Also the calculated discount amount will be shown. Original price of the product is striked and discounted value is shown.

Add Product to Cart

When add the discount applied product, it will applied throughout the process.

In cart notification also discounted value is applied.

If increase the product quantity discount value is applied based on the quantity.

Discount value is applied to the payment page.

Order is generated for the discount product.

Website order is created.
